# GachaSimulator
Siv3Dを用いた簡易的なガチャシミュレータです。

## 使い方
1. リポジトリをクローンまたはダウンロードします。
2. Siv3Dの環境をセットアップします。Siv3Dの公式ドキュメントを参照してください。
3. プロジェクトをビルドして実行します。
4. 1連または10連のガチャを引くことができます。
